Static Positioning

This is a statically positioned element, positioned normally to the document

Relative Positioning

This is a relatively positioned element. It is positioned relative to its normal position.

Absolute Positioning

This is an absolutely positioned element. It is positioned relative to nearest positioned ancestor.

Fixed Positioning

This is a fixed positioned element. It is positioned relative to the viewport. So always stays in the same place even if scrolled

Sticky Positioning

This is a sticky positioned element, it is positioned based on user's scroll. Toggles between relative based on the scroll
